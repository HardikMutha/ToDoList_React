# Simple To-Do-List

This is a Simple To-Do List Made using React.js and MUI

# Local Setup

### Requirements
-   node

### Installation

1. Clone the repo

    ```
    git clone https://github.com/HardikMutha/ToDoList_React.git
    ```

2. Install NPM packages

    ```
    npm install
    ```
3. Start

    ```
    node app.js
    ```
